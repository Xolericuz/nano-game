# SUPER PAC-MAN X | 100 LEVELS

Professional Pac-Man game with 100 levels, PWA support, and mobile controls.

## Play Online

https://Xolericuz.github.io/nano-game/

## Features

- 100 Levels
- Dash Ability (Press SHIFT)
- Mobile Controls (D-Pad + Dash Button)
- High Score (saved locally)
- PWA Support (installable on mobile)
- Offline Play
- Professional Audio Effects

## Controls

### Desktop
- Arrow Keys or W/A/S/D - Move
- SHIFT - Dash

### Mobile
- D-Pad - Move
- DASH Button - Dash

## Download APK

See [Releases](https://github.com/Xolericuz/nano-game/releases) or visit `/apk/` page.

## Tech Stack

- Pure HTML5/CSS3/JavaScript
- Canvas API
- Web Audio API
- PWA (Service Worker)
